## Key Features

Shizen UI brings modern design patterns and developer experience to Svelte 5 applications.

<CardGroup cols={2}>
  <Card title="Svelte 5 Native" icon="bolt">
    Built from the ground up using Svelte 5 runes and snippets API for optimal performance and developer experience
  </Card>
  <Card title="Tailwind CSS v4" icon="paintbrush">
    Seamlessly integrated with Tailwind CSS v4 for modern styling and customization
  </Card>
  <Card title="Compound Components" icon="layer-group">
    Flexible compound component patterns that give you full control over composition and structure
  </Card>
  <Card title="Micro-interactions" icon="wand-magic-sparkles">
    Delightful animations and micro-interactions that enhance user experience without sacrificing performance
  </Card>
  <Card title="Accessible by Default" icon="universal-access">
    ARIA-compliant form controls with keyboard navigation and screen reader support built in
  </Card>
  <Card title="TypeScript First" icon="code">
    Fully typed components with excellent IDE support and autocomplete
  </Card>
  <Card title="Customizable Variants" icon="sliders">
    Powerful variant system using tailwind-variants for consistent, customizable component styling
  </Card>
  <Card title="Minimal Bundle" icon="feather">
    Optimized for small bundle sizes with tree-shaking support and zero unnecessary dependencies
  </Card>
</CardGroup>
