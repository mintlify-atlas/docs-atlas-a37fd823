## Overview

The `@shizen-ui/styles` package provides the foundational styling system for Shizen UI. It includes CSS files, utility functions, and variant definitions built on Tailwind CSS v4.

## Installation

```bash
npm install @shizen-ui/styles
```

## Package Exports

The package provides multiple export paths:

### JavaScript/TypeScript API

```typescript
import { cn, buttonStyles, inputStyles } from '@shizen-ui/styles';
```

**Source:** `packages/styles/src/index.ts`

### CSS Exports

```typescript
// Main stylesheet
import '@shizen-ui/styles/styles';

// Individual component styles
import '@shizen-ui/styles/components/button.css';
import '@shizen-ui/styles/components/input.css';

// Theme utilities
import '@shizen-ui/styles/theme/variables.css';
import '@shizen-ui/styles/theme/utilities.css';
```

## Main Stylesheet

The main `index.css` imports Tailwind CSS v4 and all theme/component styles:

@import "./components/index.css";

## Using Styles Independently

You can use the styles package without the Svelte components:

```typescript
import { cn, buttonStyles } from '@shizen-ui/styles';
import '@shizen-ui/styles/styles';

const className = cn(
  buttonStyles({ variant: 'primary', size: 'md' }),
  'custom-class'
);
```

This is useful for:
- Building components in other frameworks (React, Vue, etc.)
- Creating custom components with Shizen UI styling
- Using only the utility functions

## TypeScript Support

All utilities and variants include full TypeScript definitions:

```typescript
import type { ButtonVariants } from '@shizen-ui/styles';
import type { ClassValue } from 'clsx';
import type { VariantProps } from 'tailwind-variants';

// ButtonVariants is inferred from buttonStyles
type ButtonProps = {
  variant?: ButtonVariants['variant'];
  size?: ButtonVariants['size'];
};
```

## Usage Example

```typescript
import { cn, buttonStyles, inputStyles } from '@shizen-ui/styles';
import '@shizen-ui/styles/styles';

// Button with variants
const button = document.createElement('button');
button.className = buttonStyles({
  variant: 'primary',
  size: 'lg'
});

// Input with custom classes
const input = document.createElement('input');
input.className = cn(
  inputStyles({ size: 'md' }),
  'w-full',
  'max-w-xs'
);

// Conditional classes
const conditionalButton = buttonStyles({
  variant: isActive ? 'primary' : 'secondary',
  size: 'md'
});
```
